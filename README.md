# Lavandaria Roupa Limpa

Site institucional da Lavandaria Roupa Limpa, desenvolvido em React com foco em apresentação de serviços, contacto e experiência moderna em desktop e mobile.

## Live
- https://lavandaria-roupalimpa.com

## Repositório
- https://github.com/Goncalompb/lavandaria-roupalimpa

## Tecnologias
- React
- Vite
- React Router
- CSS

## Funcionalidades
- Página principal com hero e secções de serviços
- Páginas de detalhe para cada serviço
- Formulário de agendamento por email
- Mapa com localização da lavandaria
- Layout responsivo (telemóvel, tablet e desktop)
- SEO base (meta tags, robots.txt, sitemap.xml)

## Como correr localmente
```bash
npm install
npm run dev
npm run build
